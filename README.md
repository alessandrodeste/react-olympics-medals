# React Olympics Medals

Exercises with react. An Olympic medal table.

## Demo

Demo on http://www.deste.org/olympics/#/

## How to use

Install packages

```bash
npm install
```

Run application with webpack-dev-server

```bash
npm run start
```

Run tests

```bash
npm run test
```

Compile for production

```bash
npm run prod
```
